### Assembly
The preliminary assembly was constructed using Hifiasm and 10x coverage Pacbio HiFi long read data and then aligned to the reference (Sbicolor_454_v3.0.1) using MUMmer. Contigs were sorted using coordinate data from MUMmer and code developed by the cooper lab at UNCC. Final assembly statistics are as follows:

Final Contig Count: 518

Final Base Pair Count: 741666270
